### Home Screen
Home screen is the first component is drawn after a succesfull login and it should be the default navigation page.

The home screen should display a list of saved catalog (From the user) and notifications.
It should allow the user navigate quickly to it's own catalog.

If there are no notifications to show, the page should display "Nada por el momento"
If the user does not have any product in it's own catalog, it should display a message saying "Empieza a vender en Alkachof".
