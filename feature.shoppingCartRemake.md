## Shopping cart remake
We are currently allowing to the shopping cart to be visible from the home screen in Alkachof
but we will limit the visibility of it.

1. From now, shopping cart will be available only when a user lands on a Catalog through the small badge of the shopping cart.
2. Adding items to a shopping cart will no longer trigger any backend calls. We will handle it exclusively using react tools. Only when a checkout has been made we will handle the backend calls
3. The cart should be fully funcionlay using mock items
